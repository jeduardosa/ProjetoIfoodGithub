# 🍔 ifoodApp

Aplicativo desenvolvido como desafio de prática de **Git e GitHub em equipe**, simulando o fluxo de trabalho colaborativo em uma empresa de software.  
O projeto consiste em um app estilo **iFood**, com foco em **branches, commits semânticos, pull requests, resolução de conflitos e revisão de código**.

---

## 🚀 Objetivo
Criar um aplicativo com 4 telas principais, simulando o fluxo de um pedido em um app de delivery.
